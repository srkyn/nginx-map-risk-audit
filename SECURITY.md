# Security

This repository is a defensive research and validation artifact. It does not include exploit payloads, crash traffic, or production probing.

## Scope

- Safe NGINX configuration review
- Local lab validation
- Splunk, Defender, and Sigma hunting notes
- Patch and exposure triage workflow

## Out Of Scope

- Exploit development
- Crash reproduction against live services
- Scanning or probing systems without authorization
- Payloads intended to trigger CVE-2026-42533

If you spot an issue in the scanner or documentation, open a GitHub issue with the smallest safe example needed to reproduce it.
